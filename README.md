# 🎓 Faculdade — Análise e Desenvolvimento de Sistemas

Este repositório reúne toda a minha trajetória acadêmica no curso de **Análise e Desenvolvimento de Sistemas**, iniciado em **2023** e concluído no final de **2025**.

Após a conclusão do curso, decidi organizar todo o conteúdo produzido ao longo da graduação com o objetivo de transformar meus estudos em um **portfólio estruturado, acessível e profissional no GitHub**.

---

## 🎯 Objetivo do Repositório

* Centralizar projetos, exercícios e materiais acadêmicos
* Demonstrar minha evolução ao longo da graduação
* Servir como base de consulta e revisão
* Apresentar minhas habilidades práticas para oportunidades profissionais

---

## 📅 Estrutura por Período
